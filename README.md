# CodeStack 🚀

A free browser extension that lets you append multiple code blocks across Jupyter Notebooks, Google Colab, ChatGPT, and GitHub into one master clipboard.

## How to Install (For Free)

1. Click the green **Code** button at the top right of this page and select **Download ZIP**.
2. Extract (unzip) the downloaded file into a folder on your computer.
3. Open Google Chrome and navigate to `chrome://extensions/`.
4. In the top-right corner, flip the **Developer mode** toggle switch to **ON**.
5. In the top-left corner, click the **Load unpacked** button.
6. Select the extracted folder containing the extension files.

## How to Use
- Open any Jupyter Notebook, Google Colab, GitHub file, or ChatGPT window.
- Click the **➕ Append** button next to any code block to add it to your sidebar stack.
- Click **📋 Copy Combined Code** to grab everything at once, separated cleanly by line breaks!
